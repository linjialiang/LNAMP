# 不需要考虑的构建选项

| configure 选项        | 描述                                 |
| --------------------- | ------------------------------------ |
| --with-cc=PATH        | 设置 C 编译器的名称。                |
| --with-cpp=PATH       | 设置 C 预处理器的名称。              |
| --with-cc-opt=OPTIONS | 设置将添加到 CFLAGS 变量的其他参数。 |
| --with-ld-opt=OPTIONS | 设置将在链接期间使用的其他参数。     |
| --with-cpu-opt=CPU    | 启用每个指定 CPU 的构建              |
