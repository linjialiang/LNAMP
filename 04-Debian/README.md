# <center>《Debian 操作系统》</center>

## Debian 是什么？

> `Debian` 是一款优秀的 `Linux` 发行版本，它遵循 GNU 开源协议，可用于服务器及桌面系统！

## Linux 又是什么?

> Linux 是一种开源电脑操作系统的内核。它是一个用 C 语言写成，符合 POSIX 标准的类 Unix 操作系统。

| 发行版 | 内核  | 应用场景     |
| ------ | ----- | ------------ |
| Debian | Linux | 服务器及桌面 |
