# MariaDB 数据类型

> 数据类型是数据的一种属性，它可以决定数据的存储格式、有效范围和相应的限制。 MariaDB 的数据类型包括:

```shell
1. 整数类型
2. 浮点数类型（不如定点数精确）
3. 定点数类型
4. 日期和时间类型（建议用时间戳）
5. 字符串类型
6. 二进制数据类型（一般用不到）
```
