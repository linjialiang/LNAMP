# 表格

> bootstrap对表格做了很大的优化，我们直接拿来就能用

## 最基本的表格元素标签

| 标签                                | 说明           |
| --------------------------------- | ------------ |
| `.container` & `.container-fluid` | bootstrap的容器 |
| `<table>`                         | 起始标签         |
| `<thead>`                         | 头标签          |
| `<tbody>`                         | 内容标签         |
| `<tr>`                            | 行标签          |
| `<th>`                            | 列标签中的标题标签    |
| `<td>`                            | 列表去中的内容标签    |

> 一个标准的表格案例：

```html
<section class="container">
    <table class="table">
      <thead class="thead-light">
        <tr>
          <th>#</th>
          <th>First</th>
          <th>Last</th>
        </tr>
      </thead>
      <tbody>
        <tr>
          <th>1</th>
          <td>Mark</td>
          <td>Otto</td>
        </tr>
        <tr>
          <th>2</th>
          <td>Jacob</td>
          <td>Thornton</td>
        </tr>
      </tbody>
    </table>
</section>
```

## 表格行高

> `bootstrap` 用 `.table-sm` 来降低表格行高

| `.table-sm`类所在位置 | 表格效果                 |
| ---------------- | -------------------- |
| `<table>`        | `整个表格`内的行高都会减低       |
| `<thead>`        | 表格中 `头标签` 内的行高会降低    |
| `<tbody>`        | 表格中 `内容标签` 内的行高会降低   |
| `<tr>`           | 表格中指定 `<tr>` 下的行高会降低 |

> bootstrap源码

```css
.table-sm th,
.table-sm td {
  padding: 0.3rem;
}
```
