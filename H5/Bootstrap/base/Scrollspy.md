# Scrollspy - 滚动监听组件

> 滚动监听组件，根据滚动位置自动更新引导导航或列表组组件，以指示视图中当前处于活动状态的链接。

## Scrollspy 几个必要条件

| 序号 | 条件描述                                                       |
| ---- | -------------------------------------------------------------- |
| 1    | 监听对象必须是 `导航组件` 或 `列表组`                          |
| 2    | `被监听内容` 需要紧接在 `监听导航组件` 下方                    |
| 3    | `被监听内容` 的父元素，需设置 `height` 和 `overflow-y:scroll;` |
| 4    | `<a>` 标签必须具有正确指向`id`的锚标记                         |
| 5    | 必须引入 bootstrap 的 js                                       |
